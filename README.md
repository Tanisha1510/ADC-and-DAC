## Experiment 7(b): Digital to Analog Converter (DAC) using Proteus
## Aim
To design and simulate a Digital to Analog Converter (DAC) circuit using Proteus and observe the conversion of digital signals into analog output voltage.</br>
## Apparatus / Software Required
•	Proteus Design Suite</br>
•	DAC IC (DAC0808)</br>
•	Resistors</br>
•	Operational Amplifier (e.g., µA741)</br>
•	Digital input switches</br>
•	Power supply</br>
## Theory
A Digital to Analog Converter (DAC) converts digital binary data into an equivalent analog voltage or current.</br>
The DAC0808 is an 8-bit DAC that produces an analog current proportional to the digital input. An operational amplifier is used to convert the output current into a voltage signal.</br>
## Features
•	8-bit resolution</br>
•	Fast conversion</br>
•	Compatible with digital systems</br>
The analog output voltage depends on the digital input combination.</br>
## Circuit Diagram
Design the circuit in Proteus using DAC0808 with:</br>
•	8 digital switches for binary input</br>
•	Op-Amp for current to voltage conversion</br>
•	Output connected to voltmeter</br>

## Procedure
1.	Open Proteus software.</br>
2.	Select components:</br>
o	DAC0808</br>
o	Operational amplifier (741)</br>
o	Resistors</br>
o	Digital switches</br>
3.	Connect binary inputs (D0–D7) using switches.</br>
4.	Connect DAC output to the op-amp circuit.</br>
5.	Connect a voltmeter at the output.</br>
6.	Run the simulation.</br>
7.	Change digital input combinations and observe the output voltage.</br>
